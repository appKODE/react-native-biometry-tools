# react-native-biometry-tools

Tools and helpers for biometry

## Installation

```sh
npm install react-native-biometry-tools
```

## Usage

```js
import BiometryTools from "react-native-biometry-tools";

// ...

const result = await BiometryTools.multiply(3, 7);
```
